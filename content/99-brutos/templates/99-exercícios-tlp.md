# Desenvolvimento

mecanismo tal da mitocôndria faz isso e aquilo etc.

Exemplo de flashcard:
```
Qual a função da mitocôndria? :: É a organela responsável pela respiração celular.
```

Fazendo a derivada:
$$\frac{d(x^4/3)}{dx} = 4x^{4-1}/3 = 4x^3/3$$
```
Qual a derivada de $\frac{d(x^4/3)}{dx}$? :: $4x^3/3$
```

Obs: remova dos blocos de aspas para usar os flashcards.
